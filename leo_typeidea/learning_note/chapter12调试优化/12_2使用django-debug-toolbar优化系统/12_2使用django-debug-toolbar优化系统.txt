12.2.1 快速配置
    1. pip install django-debug-toolbar==1.9.1
    2. setting/sevelop.py中增加
        INSTALLED_APPS += [
            'debug_toolbar',
        ]
        MIDDLEWARE += [
            'debug_toolbar.middleware.DebugToolbarMiddleware',
        ]
        INTERNAL_IPS = ['127.0.0.1']
    3. urls.py中增加
        if settings.DEBUG:
            import debug_toolbar
            urlpatterns = [
                url(r'^__debug__/',include(debug_toolbar.urls)),
            ] + urlpatterns
12.2.2 解读数据
    Version     展示项目用到的第三方库及版本
    时间         1. 服务器端执行耗时
                2. 浏览器端请求相应耗时统计
    SQL         执行了哪些sql语句
    Setting     展示目前项目的所有setting配置
    Headers     当前请求的header数据
    Request     请求的一些数据
    Static files页面用到的静态文件及目前有哪些静态文件可用
    Templetes   用到的模板及渲染模板用到的上下文变量
    Cache       缓存统计
    Signals
    Logging     通过logging模块记录了哪些日志
12.2.3 配置第三方panel
    1. djdt_flamegraph火焰图
        1. pip install djdt_flamegraph==0.2.12
        2. develop添加
         DEBUG_TOOLBAR_PANELS = {
            'djdt_framegraph.FlamegraphPanel'
        }
        3. 启动项目
        python manage.py runserver --noreload --nothreading
            --noreload      修改代码不自动重启
            --nothreading   单线程方式运行项目
    2. pympler内存占用分析
        1. pip install pympler==0.5
        2. develop配置
            INSTALLED_APPS += [
                'debug_toolbar',
                'pympler',
            ]
            DEBUG_TOOLBAR_PANELS = {
                'pympler.panels.MemoryPanel'
            }
        3. 启动项目
            python manage.py runserver
    3. line_profile 行级性能分析插件
        1. 安装
            pip install django-debug-toolbar-line-profiler==0.6.1
        2. develop配置
            INSTALLED_APPS += [
                'debug_toolbar',
                'debug_toolbar_line_profiler',
            ]
            DEBUG_TOOLBAR_PANELS = {
                'debug_toolbar_line_profiler.panel.ProfilingPanel'
            }
        3. 启动项目
            python manage.py runserver
